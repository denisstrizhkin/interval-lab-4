# Интервальный анализ данных (Лабораторная работа №4)

Проект посвящен вычислению интервальных описательных статистик и оптимизации параметров функционалов на основе коэффициента Жаккара. В качестве входных данных используются бинарные файлы диагностики томсоновского рассеяния.

## Основные возможности

Программа реализует расчет и оптимизацию параметров для четырех типов функционалов ($J_1 - J_4$):
1. **F1 (Raw)**: Средний коэффициент Жаккара для исходных интервальных выборок.
2. **F2 (Mode)**: Коэффициент Жаккара для интервальных мод выборок.
3. **F3 (MedK)**: Коэффициент Жаккара для медиан Крейновича.
4. **F4 (MedP)**: Коэффициент Жаккара для медиан Пролубникова.

Для поиска оптимальных параметров ($a$ для аддитивной модели и $t$ для мультипликативной) используется **метод золотого сечения**.

## Технологический стек

* **Rust**: Основная логика обработки данных и математические вычисления.
* **Gnuplot**: Генерация графиков зависимостей функционалов от параметров.
* **LaTeX**: Формирование итогового отчета (PDF).
* **Podman/Docker**: Изолированная среда для компиляции LaTeX.

## Инструкция по запуску

Для автоматизации всего цикла (расчет -> графики -> отчет) используйте скрипт `run_all.sh`:

```bash
chmod +x run_all.sh
./run_all.sh
```

### Предварительные условия
* Установленный **Rust** (cargo).
* Установленный **gnuplot**.
* Установленный **Podman/Docker** (для генерации PDF отчета через образ `texlive`).

## Структура проекта

* `src/main.rs` — реализация интервальной арифметики, статистик и алгоритмов оптимизации.
* `data/` — директория с исходными бинарными файлами данных.
* `report/` — исходные файлы LaTeX и сгенерированный PDF отчет.
* `plot_graphs.gp` — скрипт Gnuplot для визуализации.
* `run_all.sh` — оркестрационный скрипт.

## Результаты

Программа выводит итоговую таблицу сравнения рассчитанных параметров с контрольными значениями. Подробный анализ и графики доступны в файле `report/report.pdf`.
